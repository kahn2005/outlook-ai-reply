## 문제 정의

### 핵심 문제
Outlook을 사용하는 업무 환경에서, **수신된 이메일의 내용을 매번 읽고 직접 답변을 작성하는 데 많은 시간과 에너지가 소요된다.**

### 사용자 상황
- **도구**: Microsoft Outlook
- **언어 환경**: 일본어 및 한국어 혼용 (수신 언어에 따라 답변 언어도 달라져야 함)
- **현재 불편함**: 수신 메일의 내용을 파악하고 적절한 톤/형식으로 답변 초안을 작성하는 반복 작업

---

## 해결 아이디어

### 아이디어명
**Outlook AI 답변 자동 초안 생성기**

### 개념 설명
수신된 이메일의 내용을 AI가 자동으로 분석하여, 적절한 답변 초안을 생성하고 Outlook의 답장 작성 영역에 자동으로 채워주는 기능.

### 주요 기능 (초안)
1. **메일 내용 분석** — 수신 메일의 의도, 요청 사항, 감정 톤 파악
2. **언어 자동 감지** — 일본어/한국어 자동 판별
3. **답변 초안 생성** — 맥락에 맞는 적절한 답변 초안 작성 (동일 언어로)
4. **Outlook 자동 입력** — 답장 창에 초안 자동 삽입

---

## 기대 효과
- 이메일 답변 작성 시간 대폭 단축
- 언어 표현에 대한 심리적 부담 감소 (특히 일본어 경어체 등)
- 업무 집중도 향상
